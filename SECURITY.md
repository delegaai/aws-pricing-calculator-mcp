# Security

## Reporting vulnerabilities

Report security issues to:

```text
support@delega.ai
```

Please do not open public GitHub issues for vulnerabilities, exposed credentials, or account-specific data.

## Data and credentials

delega.ai for AWS Pricing Calculator does not require AWS credentials and does not provision AWS infrastructure.

Do not commit delega.ai API keys, bearer tokens, OAuth tokens, exported estimates containing confidential customer data, or screenshots that reveal private pricing workspaces.

## Scope

This repository contains public connection guidance for the hosted MCP server. It does not contain the hosted service backend.
