# mcpc

Use `mcpc` to inspect and test the hosted MCP server.

```bash
mcpc --version
```

Start a login flow or configure the server according to the authentication mode available to your account:

```bash
mcpc login https://calculator.delega.ai/mcp
```

Then list tools and call a small test action from a named session.

Do not paste OAuth tokens or API keys into logs, issues, or screenshots.
