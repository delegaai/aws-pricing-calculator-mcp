# Support

For setup help, billing questions, or MCP client compatibility issues, contact:

```text
support@delega.ai
```

Useful links:

- Client setup: <https://calculator.delega.ai/start/client>
- Dashboard: <https://calculator.delega.ai/app>

When reporting a client setup issue, include:

- MCP client name and version.
- Whether you used OAuth or an API key.
- The server URL you configured.
- The error message shown by the client.

Do not include access tokens, API keys, or customer-confidential estimate data.
